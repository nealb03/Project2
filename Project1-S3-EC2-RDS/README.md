\# Project 1: S3 + EC2 + RDS Architecture



\## Architecture Overview



Traditional three-tier web application architecture using AWS services:



\- \*\*Frontend:\*\* React SPA hosted on AWS S3 Static Website Hosting

\- \*\*Backend:\*\* ASP.NET Core Web API on Windows Server EC2

\- \*\*Database:\*\* MySQL 8.0 on Amazon RDS
